# AI/科技公众号图文证据稿交付模板

> 使用方式：复制本模板，替换占位内容。适合 AI 产品发布、GitHub 项目、论文、开源工具、开发者社区热点等选题。

---

# 选题标题

一句话结论：**这里直接写最重要判断。**

## 8. 正文初稿

> 这里写完整公众号正文。

要求：

- 开头不要写“随着 AI 的发展”。
- 先给结论，再补背景。
- 段落短，适合手机阅读。
- 事实、推测、观点分开。
- 对不确定信息使用“目前 / 按官方说法 / 从公开信息看 / 尚未验证”。

## 14. Review Checklist

### Fact Check

- [ ] 核心事实都有一手源
- [ ] 日期和发布主体明确
- [ ] 二手媒体没有被当成事实源
- [ ] 推测和观点已标明
- [ ] 风险边界写清楚

### Visual Check

- [ ] 每张图都有明确作用
- [ ] 截图来源、标题、日期可见
- [ ] 生成图没有冒充证据
- [ ] 移动端可读

### Writing Check

- [ ] 开头直接给重点
- [ ] 没有空泛背景
- [ ] 段落足够短
- [ ] 重点判断加粗
- [ ] 结尾有明确判断，不是鸡汤
